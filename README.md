# Word Pages to Images (Windows)

Приложение берёт входной Word-документ и создаёт новый `.docx`, где каждая исходная страница вставлена как картинка.

## Что нужно

- Windows
- Установленный Microsoft Word (обязательно)
- Python 3.10+

## Быстрый запуск

```powershell
py -3 -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python app.py
```

## Сборка EXE

```powershell
build.bat
```

После сборки готовый файл:

- `dist/WordPagesToImages.exe`

## Как использовать

1. Нажмите `Обзор...` и выберите входной файл (`.doc`, `.docx`, `.rtf`, `.odt`, `.txt`, `.html` и др.)
2. Нажмите `Сохранить как...` и укажите путь результата `.docx`
3. Нажмите `Конвертировать`

## Важно

- Для корректной конвертации документ не должен быть открыт в другом приложении.
- Качество изображений зависит от DPI (в коде сейчас 220).
